# sudokuD-WEB 🧩

A lightweight, responsive Sudoku web application built with **Python** and **Flask**, ready for one-click deployment on **Render**.

## 🚀 Features
* **Classic 9x9 Grid:** Playable directly in your browser.
* **Instant Logic:** Backend powered by Python to handle game states.
* **Responsive Design:** Works on both desktop and mobile devices.
* **Render Ready:** Includes `Gunicorn` and `requirements.txt` for easy hosting.
